# Ícones do App

Use esta pasta para ícones próprios da interface da Rádio L20.

- `SVG`: ideal para ícones simples, vetoriais, leves e fáceis de recolorir.
- `WebP`: melhor para ícones ilustrados ou detalhados.
- `PNG`: evite para ícones do app, salvo quando precisar preservar transparência ou não houver alternativa.

Mantenha os ícones consistentes com a identidade azul e amarelo do app.
